## 🛠️ Troubleshooting Quick Access

### Erori Comune

| Eroare | Soluție | Document |
|--------|---------|----------|
| Orchestrator nu pornește | npm install + rebuild | QUICK_START_GUIDE.md → Troubleshooting |
